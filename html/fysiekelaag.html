De Fysieke Laag

De fysieke laag is alles wat te maken heeft met de hardware van een apparaat. Deze laag zorgt ervoor dat gegevens kunnen worden verwerkt en verstuurd, zodat het apparaat functioneert. Zonder deze laag werkt het apparaat niet. De fysieke laag bestaat uit verschillende onderdelen die cruciaal zijn voor de werking van het apparaat.

Processor

De CPU (Central Processing Unit) is het hart van een computer. Deze zorgt ervoor dat alles snel wordt verwerkt. Taken die de gebruiker geeft aan de computer, vangt de CPU op en regelt alles op de achtergrond. De CPU is het brein van een computer, zonder dit brein is er geen werkend systeem. De processor voert miljarden taken per seconde uit, wat leidt tot de kloksnelheid.

Kloksnelheid

De kloksnelheid bepaalt hoe snel de CPU gegevens kan verwerken. Hoe hoger de kloksnelheid, hoe sneller de CPU taken kan uitvoeren. Zelf gebruik ik een Intel Core i5-12400 F, die een kloksnelheid van 2500 MHz heeft, met een maximale turbo van 4400 MHz per core. De kloksnelheid wordt gemeten in Megahertz (MHz) of Gigahertz (GHz).

Verwerkingssnelheid

Mijn CPU heeft 6 cores, wat betekent dat de processor meerdere taken tegelijk kan uitvoeren om de computer sneller te maken. De verwerkingssnelheid is afhankelijk van het aantal kernen, maar ook van de kloksnelheid. Hoe meer kernen, hoe meer taken de CPU tegelijk kan uitvoeren.

Permanente Geheugen

Permanente geheugen kan gegevens bewaren, ongeacht of de computer aan- of uitstaat. Dit geheugen wordt gebruikt voor lange termijn opslag van gegevens. Voorbeelden hiervan zijn flash geheugen, Solid State Drives (SSD) en harde schijven.

Randapparatuur

Randapparatuur is hardware die je kunt verbinden met je computer om extra functies toe te voegen. Voorbeelden van randapparatuur zijn: printers, toetsenborden en muizen. Deze apparaten maken het gebruik van de computer makkelijker en handiger.
