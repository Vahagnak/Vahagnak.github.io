Welkom op mijn website!

In deze code wordt de afbeelding als achtergrond ingesteld met behulp van CSS. Het body::before pseudo-element voegt een donkere overlay toe om de afbeelding te verduisteren. Je kunt de rgba(0, 0, 0, 0.5)-waarde aanpassen om de donkerte aan te passen. Een waarde van 0 betekent volledig transparant, en een waarde van 1 betekent volledig ondoorzichtig. Experimenteer met verschillende waarden om het gewenste effect te bereiken.
