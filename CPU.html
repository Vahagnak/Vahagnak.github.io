De Central Processing Unit [CPU]

De Central Processing Unit, ofwel CPU, is het brein van een computer en beheert de gehele werking door snelle basisberekeningen uit te voeren. Gewoonlijk bevindt de CPU zich in het midden-bovenin, naast het RAM-geheugen en onder een CPU-koeler. Instructies voor de processor worden vaak opgeslagen en opgehaald uit het Random Access Memory (RAM) door de CPU zelf. De oplossingen worden vervolgens teruggeplaatst in het RAM. Indien nodig worden de instructies naar een eindbestemming gestuurd, zoals een grafische kaart of een opslagmedium zoals een harde schijf of SSD.

Snelheid

De snelheid van de processor wordt bepaald door de kloksnelheid, het aantal cores en het aantal threads. Moderne processors hebben meerdere kernen (cores), waarbij elke core één of twee rekentaken tegelijk kan uitvoeren, bekend als threads. Meestal wordt één kern aan één programma toegewezen, tenzij het programma gebruik kan maken van meerdere kernen, zoals bij games en bepaalde werkprogramma's zoals Adobe Creative Suite. De kloksnelheid geeft aan hoeveel berekeningen er per seconde door één core kunnen worden uitgevoerd. Een core die op 5 GHz werkt, voert bijvoorbeeld meer berekeningen uit per klokcycli dan een core die op 3 GHz draait.

De rol van de CPU

De CPU bepaalt der theoretische maximale snelheid van een computer. Voor processen die gebruikmaken van de meerdere kernen van de CPU is voldoende vermogen essentieel. Een krachtigere CPU draagt ook bij aan een langere levensduur van de computer, aangezien de CPU over het algemeen niet de bottleneck is die snelheidsverschillen tussen onderdelen veroorzaakt.
